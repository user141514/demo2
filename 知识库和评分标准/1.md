**行动学习作业汇报量化评分标准**

**（第一次课程《趋势变局下MBA管理者的认知升级》专用）**

**一、评分基础规则**

**1.1 计分范围**

**各子维度评分区间为0-10分，最小计量单位为0.1分，严格按锚定标准判定区间。**

**1.2 评分锚定区间**

| **区间** | **等级** | **判定标准** |
| :-: | :-: | :-: |
| **9.0-10.0分** | **卓越** | **表现远超预期，可作为标杆示范** |
| **7.5-8.9分** | **优秀** | **表现良好，有少量可改进空间** |
| **6.0-7.4分** | **良好** | **基本达标，存在明显改进空间** |
| **4.0-5.9分** | **合格** | **勉强达标，需要较大幅度改进** |
| **0.0-3.9分** | **不合格** | **未达标，需要重新学习并提交** |


**1.3 核心原则**

**以"可观测行为、可量化结果、可验证价值"为核心打分依据，区间差异聚焦"问题穿透深度、方案设计精度、落地可行性、表达呈现质量"，确保打分公平、统一、可追溯。**

**1.4 权重说明**

**一级维度权重为作业评价80%、呈现效果评价20%。**

**最终总分 = Σ（子维度得分 × 子维度权重 × 一级维度权重）**

**1.5 评分维度总览**

| **一级维度** | **权重** | **子维度** | **子维度权重** |
| :-: | :-: | :-: | :-: |
| **作业评价** | **80%** | **直面问题** | **20%** |
|  |  | **创新构想** | **15%** |
|  |  | **结构性方法** | **15%** |
|  |  | **可操作性** | **30%** |
| **呈现效果评价** | **20%** | **表达清晰** | **10%** |
|  |  | **回答问题** | **5%** |
|  |  | **时间管理** | **5%** |


**二、作业评价（权重80%，子维度总分10分）**

**2.1 直面问题（权重20%，核心：问题穿透深度）**

**评估学员是否精准揭示了个人发展中的核心问题和关键障碍，是否停留在表面文章，论据是否充分具有说服力。**

| **区间** | **评分标准** |
| :-: | :-: |
| **9.0-10.0 卓越** | **精准揭示个人发展核心、关键问题，完全不避重就轻，论据数据详实、逻辑严密，具备极强的说服力，能直接支撑后续IDP设计。** |
| **7.5-8.9 优秀** | **清晰识别核心问题，未停留于表面现象，论据充分合理，说服力较强，能有效支撑IDP的针对性设计。** |
| **6.0-7.4 良好** | **能识别主要问题，未刻意回避核心矛盾，论据基本合理，具备基本说服力，但问题挖掘不够深入，存在一定的表面化倾向。** |
| **4.0-5.9 合格** | **问题识别模糊，存在避重就轻倾向，论据单薄，说服力不足，对核心问题的揭示不够明确，难以支撑IDP设计。** |
| **0.0-3.9 不合格** | **未揭示核心问题，完全停留在表面描述，刻意回避关键矛盾，论据缺失或无效，无任何说服力。** |


**【评分指引】**

**重点关注：是否运用了测评工具（商业综合推理、管理技能、管理个性、管理风格、职业锚、组织忠诚度）的结果进行自我认知分析。**

**重点关注：自我评估是否客观深入，是否识别了自身的优势、短板和发展瓶颈。**

**重点关注：是否结合了外部环境趋势（PEST分析、行业发展趋势）进行环境评估。**

**重点关注：是否避免了"自我美化""归因外部""浅尝辄止"等常见误区。**

**2.2 创新构想（权重15%，核心：方案差异化价值）**

**评估学员是否结合作业要求和所学知识（ASTRAL领导力模型、人才管理理论等），针对个人发展问题，大胆构思和创新，提出突破性的IDP方案。**

| **区间** | **评分标准** |
| :-: | :-: |
| **9.0-10.0 卓越** | **结合作业要求、所学知识与个人实际，提出突破性创新构想，方案打破常规、具备显著挑战性，能带来超出预期的价值提升。** |
| **7.5-8.9 优秀** | **能结合所学知识与个人实际，提出明确的改进/优化构想，创新性与挑战性较强，能带来明显的价值提升，符合作业要求。** |
| **6.0-7.4 良好** | **能结合作业要求提出基本构想，有一定的优化意识，但创新性不足，整体偏向常规思路，挑战性较弱，仅能满足基本要求。** |
| **4.0-5.9 合格** | **构想照搬常规思路，无任何创新或改进思考，与作业要求结合度低，无差异化价值，仅能勉强完成基础任务。** |
| **0.0-3.9 不合格** | **无明确构想，或构想脱离作业要求、所学知识与个人实际，无任何价值，甚至不符合作业基本要求。** |


**【评分指引】**

**重点关注：IDP方案是否体现了对ASTRAL领导力模型的深度理解和应用。**

**重点关注：是否设计了创新性的发展路径和学习方式（如轮岗、项目历练、AI辅助学习等）。**

**重点关注：方案是否充分考虑了中集车辆第三次创业背景下的组织发展需求。**

**重点关注：是否体现了"从我的世界到我们的世界"的思维转变，将个人发展与组织发展相结合。**

**2.3 结构性方法（权重15%，核心：方案严谨性）**

**评估学员是否选择及应用恰当的结构化方法进行IDP的设计，使得方案严谨、系统、科学。**

| **区间** | **评分标准** |
| :-: | :-: |
| **9.0-10.0 卓越** | **精准选择并应用恰当的结构化方法设计方案，逻辑链条完整闭环，方案严谨、系统、科学，无任何逻辑漏洞，体现极强的系统性思维。** |
| **7.5-8.9 优秀** | **能正确选择并应用合适的结构化方法，方案逻辑清晰、结构完整，无明显逻辑漏洞，严谨性与系统性较强。** |
| **6.0-7.4 良好** | **能选用基本的结构化方法设计方案，方案结构基本完整，逻辑基本通顺，但方法应用不够熟练，存在轻微的逻辑衔接问题。** |
| **4.0-5.9 合格** | **未选用合适的结构化方法，或方法应用生硬、与方案不匹配，方案结构松散，逻辑链条存在断裂，严谨性不足。** |
| **0.0-3.9 不合格** | **未使用任何结构化方法，方案无逻辑、无结构，内容混乱，无法体现系统性与科学性。** |


**【评分指引】**

**重点关注：是否正确运用了IDP七步法（自我评估→环境评估→职业选择→确定目标→行动计划→行动实施→评估反馈）。**

**重点关注：是否遵循了IDP四模块结构（现状→目标→能力→计划）。**

**重点关注：是否运用了7-2-1学习法则设计学习活动（70%岗位学习+20%人际学习+10%正式学习）。**

**重点关注：方案的各个模块之间是否逻辑自洽、环环相扣。**

**2.4 可操作性（权重30%，核心：方案落地可行性）**

**评估学员的IDP方案是否有针对性、可行性和实操性，是否能真正落地执行。**

| **区间** | **评分标准** |
| :-: | :-: |
| **9.0-10.0 卓越** | **解决方案针对性极强，具备高度可行性与实操性，关键步骤清晰、资源要求明确，可直接落地推进，无明显落地障碍。** |
| **7.5-8.9 优秀** | **解决方案针对性明确，可行性与实操性较强，关键步骤清晰，资源要求合理，稍加完善即可落地推进。** |
| **6.0-7.4 良好** | **解决方案有一定针对性，具备基本可行性，但关键步骤不够细化，资源要求不够明确，落地存在一定难度，需大幅完善。** |
| **4.0-5.9 合格** | **解决方案针对性较弱，可行性不足，关键步骤模糊，资源要求不清晰，难以落地推进，仅停留在概念层面。** |
| **0.0-3.9 不合格** | **解决方案无针对性、无可行性，完全脱离实际，无法落地，甚至会带来负面风险。** |


**【评分指引】**

**重点关注：IDP是否明确了具体的行动步骤、时间节点、责任人和交付物。**

**重点关注：是否设定了可衡量的阶段性目标（1年/2年/3年）和最终目标。**

**重点关注：是否明确了资源支持（辅导/轮岗/项目/培训等）和评价方式（测评/考核/行为记录等）。**

**重点关注：方案是否符合中集车辆的组织能力和文化特点。**

**重点关注：是否设计了评估与反馈闭环机制（STEP7）。**

**三、呈现效果评价（权重20%，子维度总分10分）**

**3.1 表达清晰（权重10%，核心：信息传递效率）**

**评估学员是否清晰、准确地陈述IDP报告，使听众能快速理解核心内容。**

| **区间** | **评分标准** |
| :-: | :-: |
| **9.0-10.0 卓越** | **陈述逻辑清晰、表达精准专业，重点突出、层次分明，能高效传递核心信息，听众可快速理解报告核心内容，无任何歧义。** |
| **7.5-8.9 优秀** | **陈述清晰准确，表达流畅，重点明确，层次较分明，能有效传递核心信息，听众无理解障碍。** |
| **6.0-7.4 良好** | **陈述基本清晰，表达较流畅，重点较明确，能传递主要信息，但存在少量表述模糊或逻辑衔接不畅的问题。** |
| **4.0-5.9 合格** | **陈述不够清晰，表达存在卡顿或歧义，重点不突出，信息传递效率低，听众需反复理解才能获取关键内容。** |
| **0.0-3.9 不合格** | **陈述混乱、表达不清，逻辑完全断裂，无法传递有效信息，听众无法理解报告内容。** |


**【评分指引】**

**重点关注：PPT结构是否清晰，是否遵循了"自我认知→环境评估→职业目标→能力发展→行动计划→评估反馈"的逻辑主线。**

**重点关注：语言表达是否简洁有力，避免冗长和重复。**

**重点关注：数据可视化是否清晰易懂，测评结果的呈现是否恰当。**

**3.2 回答问题（权重5%，核心：互动响应质量）**

**评估学员在答辩环节回答问题的简练准确程度。**

| **区间** | **评分标准** |
| :-: | :-: |
| **9.0-10.0 卓越** | **回答问题简练、精准，直击核心，能快速响应评委问题，补充信息详实且贴合问题，体现对内容的深度掌握。** |
| **7.5-8.9 优秀** | **回答问题简练准确，能有效回应评委问题，补充信息合理，体现对内容的良好掌握。** |
| **6.0-7.4 良好** | **能回答评委问题，表述基本准确，但不够简练，存在少量冗余信息，基本体现对内容的掌握。** |
| **4.0-5.9 合格** | **回答问题不够准确，表述冗长或偏离问题核心，难以有效回应评委提问，对内容掌握不足。** |
| **0.0-3.9 不合格** | **无法回答评委问题，或回答完全偏离问题、表述混乱，无法体现对内容的掌握。** |


**【评分指引】**

**重点关注：是否准确理解评委问题的核心意图。**

**重点关注：回答是否简洁有力，不绕弯子。**

**重点关注：面对质疑时是否能保持冷静、有理有据地回应。**

**3.3 时间管理（权重5%，核心：流程管控能力）**

**评估学员是否在规定的时间内完成成果呈现。**

| **区间** | **评分标准** |
| :-: | :-: |
| **9.0-10.0 卓越** | **严格在规定时间内完成成果呈现，时间分配合理，无超时或提前结束的情况，节奏把控精准。** |
| **7.5-8.9 优秀** | **在规定时间内完成呈现，时间分配较合理，无严重超时或提前结束（误差≤30秒），节奏把控较好。** |
| **6.0-7.4 良好** | **基本在规定时间内完成呈现，存在轻微超时或提前结束（误差≤1分钟），时间分配存在少量不合理之处。** |
| **4.0-5.9 合格** | **呈现严重超时或提前结束（误差\>1分钟），时间分配混乱，影响整体呈现效果。** |
| **0.0-3.9 不合格** | **未在规定时间内完成呈现，超时严重或提前结束大量内容，无法完整呈现成果。** |


**【评分指引】**

**重点关注：时间分配是否与内容重要性匹配（自我认知/环境评估/职业目标/能力发展/行动计划/评估反馈）。**

**重点关注：是否在规定时间内完成了核心内容的陈述。**

**四、打分应用规则与说明**

**4.1 维度独立评分**

**评委需按上述分阶标准，对每个子维度单独打出0-10分制分数（保留1位小数）。**

**4.2 权重折算规则**

**最终总分 = Σ（子维度得分 × 子维度权重 × 一级维度权重），各一级维度得分占比严格遵循80%/20%的权重设置。**

**4.3 打分核心原则**

**以汇报中可验证的具体案例、数据、方案细节为核心依据，优先量化表现，避免主观印象分。**

**4.4 区间判定边界**

**当表现同时满足多个区间特征时，以"更高标准的核心特征是否达成"为判定依据，优先匹配更贴合的高阶区间。**

**4.5 分数精度要求**

**尽量避免整数打分，最小计量单位为0.1分，确保打分的精准度与差异化。**

**五、评分计算示例**

**假设某学员汇报的各子维度得分如下：**

| **子维度** | **得分** | **子维度权重** | **一级维度权重** | **加权得分** |
| :-: | :-: | :-: | :-: | :-: |
| **直面问题** | **8.5** | **20%** | **80%** | **1.360** |
| **创新构想** | **7.8** | **15%** | **80%** | **0.936** |
| **结构性方法** | **8.2** | **15%** | **80%** | **0.984** |
| **可操作性** | **8.0** | **30%** | **80%** | **1.920** |
| **表达清晰** | **8.5** | **10%** | **20%** | **0.170** |
| **回答问题** | **8.0** | **5%** | **20%** | **0.080** |
| **时间管理** | **9.0** | **5%** | **20%** | **0.090** |


**最终总分 = 1.360 + 0.936 + 0.984 + 1.920 + 0.170 + 0.080 + 0.090 = 5.540（满分10分）**

**按照百分制换算：5.540 × 10 = 55.4分**

**六、作业评估维度与汇报PPT框架对照**

| **PPT模块** | **权重** | **对应评分维度** | **核心要求** |
| :-: | :-: | :-: | :-: |
| **职业发展现状** | **15%** | **直面问题** | **运用测评工具进行自我认知，明确当前职务级别、工作职责、管理范畴** |
| **外部环境评估** | **15%** | **直面问题+结构性方法** | **运用PEST框架、三层趋势分析进行环境评估，识别机会与威胁** |
| **职业目标设定** | **20%** | **结构性方法+创新构想** | **设定1/2/3年职业发展目标，链接组织发展目标（中集车辆第三次创业）** |
| **能力发展计划** | **30%** | **创新构想+可操作性** | **基于ASTRAL模型确定3项重点发展能力，运用7-2-1法则设计学习活动** |
| **行动计划与评估** | **15%** | **可操作性** | **明确行动步骤、时间节点、资源支持、评价方式，设计评估反馈闭环** |
| **总结与反思** | **5%** | **直面问题+创新构想** | **核心收获、方案优化方向、对组织协同的思考** |


**七、学员常见误区及扣分指引**

| **常见误区** | **表现** | **扣分维度** | **扣分幅度** |
| :-: | :-: | :-: | :-: |
| **自我美化** | **过度强调优势，回避短板和不足** | **直面问题** | **-1.0至-2.0** |
| **归因外部** | **将问题归咎于外部环境，缺乏自我反思** | **直面问题** | **-1.0至-2.0** |
| **浅尝辄止** | **停留在表层分析，未深入挖掘根源** | **直面问题** | **-1.0至-2.0** |
| **照搬理论** | **直接套用课程理论，未结合个人实际** | **创新构想+可操作性** | **-0.5至-1.5** |
| **方案空泛** | **只有框架没有细节，缺乏具体行动步骤** | **可操作性** | **-1.5至-3.0** |
| **忽视反馈** | **未设计评估与反馈闭环机制** | **可操作性** | **-0.5至-1.5** |
| **时间失控** | **汇报超时或严重提前结束** | **时间管理** | **-1.0至-3.0** |
| **答辩失当** | **答非所问或过度辩解** | **回答问题** | **-0.5至-2.0** |
| **目标脱节** | **个人目标与组织发展目标脱节** | **创新构想** | **-0.5至-1.5** |
